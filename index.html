Flexbox: flex-grow & flex-shrink demo

Resize the window (or view on different widths) to see how boxes grow and shrink. Each box has a flex-basis, plus custom flex-grow and flex-shrink values.

Box A flex: 2 1 150px
grow = 2, shrink = 1
Box B flex: 1 2 220px
grow = 1, shrink = 2
Box C flex: 0 1 120px
grow = 0, shrink = 1

Notes:
- flex-grow controls how extra space is distributed (larger number = takes more).
- flex-shrink controls how boxes shrink when space is limited (larger number = shrinks more).
